Detailed Description

Extension of the Command class that can be paused at any given time. When paused, the internal timer does not increment. When switching between paused or resumed the respective onPause and onResume methods are calld. When paused, the execute method does not get called

Constructor & Destructor Documentation

PausableCommand::PausableCommand ( )

Extension of the Command class that allows itself to be paused Uses a pausable timer for getTime rather than millis like Command Methods onPause and onResume get called once when the state of the robot switches to the respective state. Any time based commands will behave as expected, even when paused
